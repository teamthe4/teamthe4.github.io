Great Features

Fully Responsive

Droop menu adapts on any screen size and fully optimized to work well across an array of devices ranging from mobile, tablets and desktops

Standard Markup

The menu is built with modern standard web technologies using HTML5 + CSS3 and progressively enhaced with JavaScript using the jQuery framework.

Animations

Use CSS3 animation effects for an enhanced desktop experience then switch to more subtle toggle animations on mobile and touch enabled devices

Unlimited Levels

You can easily create unlimited dropdown levels for any menu. Dropdowns can be styled in terms of sizes, direction as well as general content

Mega Menus

With a full 12 column grid system built in and custom size individual layout controls for each menu item, it's easy to build mega menu layouts.

Menu Tabs

Droopmenu includes vertical and horizontal tabs for your content. The tabs automatically switch to accordions on mobile without any extra markup

Themes & Skins

30+ beautifully crafted menu templates and themes to choose from. Each template can easily be customized and integrated with great ease.

Vertical & Horizontal

Droop mega menu allows you to create both horizontal and vertical mega menus with support for touch screens and mobile devices.

Multi Position

The menu supports multiple positions which you can choose from. Set the menu depending on preference either TOP, BOTTOM, LEFT, RIGHT of the screen

Toggle Off Canvas

The menu supports both normal mobile toggle and app style left and right off canvas menus on mobile devices with native app experience

Fixed Sticky

The menu supports both fixed position menus that stay fixed on page as well as sticky menus which adjust positioning on page scroll

Beautiful UI/UX

Droopmenu mega menu is built with modern web design in mind with a minimalistic appealing look that can match any moden website

Vector Icons

The menu comes with a full set of ionic framework icons - in addition custom styled social network icons have been icluded for all the major social networks!

Menu Alignment

You can align the menu Left, Center, Right and Left-Right depending on your preferred style which is very handy in matching your design

Full Width/Boxed

Droopmenu mega menu supports both full width stretched layout and boxed layouts designs. Fully customize the menu width to match your application

Basic Scollspy

Included in the package are more practical examples such as single page navigation, show menu on scroll-up among many other practical use cases

Overlay Search

Droop mega menu includes a beautiful modern full animated screen overlay search bar - with slide up / down and ripple effect animations

Visual Arrows

Need to know which menu you hovered, we included a set of beautiful animated CSS3 indicatior arrows that makes this menu look sleek

Accessibility

Droopmega menu follows basic accessibility standards such as keyboard access using the TAB KEY on desktops and WAI-ARIA semantics

Cross Browser

Droop mega menu is compatible with all the major desktop and mobile browsers including Chrome, Firefox, Safari, Opera, Internet Explorer and Edge.

Easy to use

Droopmenu mega menu is very simple and easy to setup in your own web application for anyone with or without basic programming skills
